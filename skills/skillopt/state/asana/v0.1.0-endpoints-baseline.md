# Asana REST API v1.0 — Endpoint Reference

Base URL: `https://app.asana.com/api/1.0`
Auth: `Authorization: Bearer <PAT>` (Personal Access Token from Developer Console)
Content-Type: `application/json`
Pagination: cursor-based via `limit` and `offset` params
Rate limits: cost-based; honor `Retry-After` header on HTTP 429

---

## Workspaces

| Method | Path | Description |
|--------|------|-------------|
| GET | `/workspaces` | List workspaces |
| GET | `/workspaces/:gid` | Get workspace |

Scope: `workspace:read`

---

## Projects

| Method | Path | Description |
|--------|------|-------------|
| GET | `/projects` | List projects. Params: `workspace`, `team`, `archived`, `opt_fields` |
| GET | `/projects/:gid` | Get project |
| POST | `/projects` | Create project. Body: `{name, workspace, team?, notes?, due_on?, color?, owner?}` |
| PUT | `/projects/:gid` | Update project |
| DELETE | `/projects/:gid` | Delete project |
| GET | `/projects/:gid/tasks` | Tasks in project |
| GET | `/projects/:gid/sections` | Sections in project |

Scopes: `projects:read`, `projects:write`

---

## Tasks

| Method | Path | Description |
|--------|------|-------------|
| GET | `/tasks` | List tasks. Params: `project`, `section`, `assignee`, `completed_since`, `opt_fields` |
| GET | `/tasks/:gid` | Get task with `opt_fields` |
| POST | `/tasks` | Create task. Body: `{name, notes?, projects?, assignee?, due_on?, start_on?, parent?, followers?, memberships?}` |
| PUT | `/tasks/:gid` | Update task. Body: `{name?, notes?, assignee?, due_on?, start_on?, completed?, approval_status?}` |
| DELETE | `/tasks/:gid` | Delete task |
| GET | `/tasks/:gid/subtasks` | List subtasks |
| POST | `/tasks/:gid/addDependencies` | Add deps: `{dependencies: [gids]}` |
| POST | `/tasks/:gid/removeDependencies` | Remove deps: `{dependencies: [gids]}` |
| POST | `/tasks/:gid/addFollowers` | Add followers: `{followers: [gids]}` |
| POST | `/tasks/:gid/removeFollowers` | Remove followers: `{followers: [gids]}` |
| POST | `/tasks/:gid/addProject` | Add to project: `{project, section?}` |
| POST | `/tasks/:gid/removeProject` | Remove from project: `{project}` |

Scopes: `tasks:read`, `tasks:write`

## Sections

| Method | Path | Description |
|--------|------|-------------|
| GET | `/projects/:gid/sections` | List sections in project |
| GET | `/sections/:gid` | Get section |
| POST | `/projects/:gid/sections` | Create section: `{name, insert_before?, insert_after?}` |
| PUT | `/sections/:gid` | Update section name |
| DELETE | `/sections/:gid` | Delete section (must be empty; last section cannot be deleted) |
| POST | `/sections/:gid/addTask` | Add task to section: `{task, insert_before?, insert_after?}` |

## Common opt_fields

- **Tasks:** `name,notes,assignee,due_on,start_on,completed,completed_at,created_at,modified_at,parent,projects,tags,followers,dependencies,dependents,custom_fields`
- **Projects:** `name,notes,workspace,team,due_on,start_on,color,owner,members,archived,custom_fields,current_status`
- **Users:** `name,email,workspaces,photo`
- **Portfolios:** `name,workspace,color,owner,custom_fields`
- **Goals:** `name,notes,workspace,team,owner,due_on,start_on,metric,status,followers`

## Error Codes

| Status | Meaning |
|--------|---------|
| 400 | Bad request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not found |
| 429 | Rate limited (check `Retry-After`) |
| 500 | Server error |
